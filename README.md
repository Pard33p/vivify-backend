# vivify-backend

## Installation Instructions

- Make sure that you have the anaconda environment installed.
- `git clone https://github.com/horcruxxx/vivify-backend`
- `cd vivify-backend`
- `make create_env`
- `conda activate vivify_backend`
- `make sandbox`
- `sandbox/manage.py runserver`
